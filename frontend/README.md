# Frontend - Authentication System

## نبذة
هذا المجلد يحتوي على الواجهة الأمامية React/Next.js لنظام المصادقة المتكامل، يدعم المصادقة، تعدد الخطوات، وRTL، مع تصميم عصري وداكن.

## التقنيات المستخدمة
- Next.js 14
- React
- TypeScript
- Tailwind CSS
- Material UI (MUI)
- Emotion (مكتبة تنسيقات لـ MUI)
- Axios (طلبات HTTP)
- دعم RTL كامل وواجهات عربية بالكامل

## طريقة التشغيل
1. تأكد من وجود Node.js و npm
2. ثبت الحزم:
   ```bash
   npm install
   ```
3. يمكنك تشغيل التطبيق:
   ```bash
   npm run dev
   ```
4. أدخل على: http://localhost:3000

## ملفات مهمة
- `app/` : جميع الصفحات والمكونات الرئيسية
- `components/ui/` : العناصر المشتركة وقوالب الحقول والأزرار
- `app/layout.tsx` : الإعداد العام للواجهة، اللغة، الاتجاه والنُسق
- `public/logo.png` : شعار الموقع الذي يظهر في التبويب
